Detailed Description

Centralised version and product-identity macros for CANoScope.

A single source of truth for the application version string, product name, vendor, and contact details. These macros are consumed by the command-line --version handler (main), the GTK About dialog (gui_show_about_dialog), and the Debian packaging metadata.
