Detailed Description

Focused regression tests for the Bit Analysis engine and DBC promotion.

Builds a set of deterministic synthetic CAN traces that exercise bit extraction, target filtering, baseline capture, candidate scanning, signedness, Motorola conversion, lag detection, counter/checksum/timestamp heuristics, multiplexer evidence, validation segments, JSON persistence, and DBC round-trips. The test binary is intentionally separate from the GTK UI so it can run in CI and on development machines without CAN hardware.
